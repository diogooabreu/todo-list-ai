# Project Requirements Document (PRD) - TodoList

**Project:** TodoList (Study to AI-assisted engineering)  
**Version:** 0.1  
**Status:** Started

---

## Objective
Manage daily tasks safely

## Main features
- The user must be able to register and login
- The user should be able to create, list, edit and delete their own taks
- The user cannot view or edit other users' tasks
- Each task should have a title, an optional description, and a status of "completed"
- The user should be able to create, list, edit and delete categories to organize their tasks
- Each task can optionally be associated with a category
